Privacy Policy

LinnEshIn Oura Bridge Privacy Policy

LinnEshIn Oura Bridge is a personal-use integration operated by a single individual for private wellness, planning, and research support.

What data is accessed

  • Sleep summaries
  • Readiness summaries
  • Activity summaries
  • Related derived wellness metrics authorized through the Oura API

How data is used

  • To display personal recovery and activity indicators in the LinnEshIn local system
  • To support personal planning, journaling, and private self-tracking workflows
  • To compare Oura metrics with other locally recorded personal data

Data sharing

No Oura data is sold, shared publicly, or provided to third parties, except for the data exchange required between the user and the official Oura API.

Data storage

Data fetched through the Oura API is stored locally by the user for personal use. The app is not intended as a public service or multi-user platform.

Contact

Questions about this personal integration can be directed to the contact email registered with the associated Oura API application.
